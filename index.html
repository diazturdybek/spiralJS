SpiralJS - can help create beautiful spiral animation!

Documentation

Copy-paste the <script> into your <body> end, before all other scripts to load our library. Also create a <div> element inside which <canvas> element. <canvas> will take the size of the parent, so to avoid bugs, you need to set the canvas-wrapper style to { position: relative; } 

			
<!DOCTYPE html>
<html lang="en">
<head>
	<meta charset="UTF-8">
	<meta name="viewport" content="width=device-width, initial-scale=1.0">
	<title>Document</title>
	<style type="text/css">
		.canvas-wrapper {
			position: relative;
			width: 100vw;
			height: 100vh;
		}
	</style>
</head>
<body>
	<div class="canvas-wrapper">
		<canvas id="spiralCanvas"></canvas>
	</div>
	<!-- Require library -->
	<script src="path/to/spiral.js"></script>
</body>
</html>

Initialization

Paste the spiralJS.init(target, [options]); into your script. 

			
				spiralJS.init(document.getElementById('spiralCanvas'), [ config array, optional]);

Options 

[lineSize] - sets line size
[lineColor] - sets line color
[angle] - sets the angle, which adjusts the distance between the lines
[anglePitch] - sets the distance between pixels, The closer to zero, the more it looks like a spiral.
[FPS] - sets framerate
[background] - sets background color
[rotatePitch] - sets rotate speed
